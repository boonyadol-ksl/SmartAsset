# สรุปรีวิวระบบจัดการทรัพย์สิน (Asset Management System) — รวม 2 ไฟล์

## ภาพรวมระบบ
ระบบเป็นแพลตฟอร์มเว็บสำหรับบริหาร “ทรัพย์สินองค์กร” ครอบคลุมตั้งแต่นำเข้าข้อมูลทรัพย์สิน, ตรวจนับภาคสนามด้วย QR Code, การรายงาน/วิเคราะห์ และการพิมพ์เอกสาร โดยมีจุดเด่นด้านความโปร่งใส ตรวจสอบย้อนหลังได้ (Audit Trail) และรองรับการใช้งานระดับองค์กร

## คุณสมบัติหลักระดับองค์กร (จากรีวิวฉบับอัปเดต)
- **รองรับ Multi-Site (หลายโรงงาน/หลายไซต์)**: แยกข้อมูลและสิทธิ์ตาม Site/Plant อย่างชัดเจน ผู้ใช้เห็นเฉพาะไซต์ที่ได้รับสิทธิ์ (RBAC แบบแยกตามโรงงาน)
- **รองรับการทำงานแบบหลายปี (Multi‑Year / รอบตรวจนับรายปี)**: เปิดรอบตรวจนับตามปี (Fiscal Year) ได้ พร้อม **Deadline** ต่อรอบ และติดตามความคืบหน้า
- **Import แบบ Incremental**: นำเข้าหรืออัปเดตทรัพย์สินปีใหม่ได้โดยไม่กระทบ Asset ID เดิม
- **ระบบ Login ระดับองค์กร**: รองรับ **Active Directory (AD) / Single Sign‑On** (และยังมี Local login เป็นทางเลือกสำรอง)
- **Auditor Performance Scoring**: คำนวณคะแนนการตรวจนับอัตโนมัติหลังจบงาน (เต็ม 100) เพื่อใช้ประเมินผลงานอย่างยุติธรรม

## โมดูล/หน้าจอสำคัญ (สรุปรวม)
### 1) Dashboard (dashboard.php)
- แสดงภาพรวมแบบเรียลไทม์: จำนวน/สถานะทรัพย์สิน, มูลค่ารวม
- แสดง **Progress การตรวจนับ** และ (ฉบับอัปเดต) **Deadline ที่เหลือ**
- กราฟสรุป เช่น Donut/Bar และสรุปตาม Asset Class
- (ฉบับอัปเดต) เพิ่มตัวกรองตาม **Site** และ **ปี**

### 2) รายการทรัพย์สิน (asset-list.php)
- ค้นหาและกรองข้อมูล (Plant/Site, Cost Center, Department, Status และในฉบับอัปเดตเพิ่ม **Year**)
- ส่งออกเป็น Excel/CSV
- (ฉบับอัปเดต) เน้นการแสดงข้อมูลเฉพาะ Site ที่ได้รับสิทธิ์

### 3) รายละเอียดทรัพย์สิน (asset-detail.php)
- แสดงข้อมูลทรัพย์สินรายชิ้น + แกลเลอรีรูป
- Timeline ประวัติการตรวจนับย้อนหลัง (ระบุ 3 ปีในรีวิวเดิม)
- จำกัดสิทธิ์แก้ไขเฉพาะ Admin

### 4) สร้าง/จัดการรอบตรวจนับ (manage_audit.php)
- สุ่มทรัพย์สินตามเปอร์เซ็นต์, มอบหมายงานตามโซน/ไซต์ และผู้ตรวจสอบ
- ป้องกันตรวจซ้ำ + ตรวจสอบภาระงาน
- (ฉบับอัปเดต) เลือก **ปีการตรวจนับ** และกำหนด **วันสิ้นสุดรอบ/Deadline**

### 5) ตรวจนับภาคสนาม (inventory.php)
- สแกน QR Code เพื่อดึงข้อมูลทรัพย์สิน
- ทำงานตามงาน/โซนที่ได้รับมอบหมาย
- เซ็นชื่อดิจิทัล 3 ฝ่าย (Auditor, Asset Officer, Department Head)
- พิมพ์รายงานสรุปได้ทันที (จากรีวิวเดิม)
- (ฉบับอัปเดต) แสดง Deadline ชัดเจน + คำนวณคะแนนอัตโนมัติเมื่อส่งงานเสร็จ

### 6) นำเข้าทรัพย์สิน (asset-import.php)
- รองรับ Excel/CSV พร้อมแสดงความคืบหน้าการอัปโหลด
- มีบันทึกประวัติการนำเข้า (Logs) (จากรีวิวเดิม)
- (ฉบับอัปเดต) ระบุ **Site ที่นำเข้า** และรองรับ Import แบบ Incremental

### 7) พิมพ์การ์ด/QR (asset-print.php)
- พิมพ์แบบ A4 (1 ทรัพย์สินต่อ 1 หน้า)
- สร้าง QR Code อัตโนมัติ + ช่องเซ็นรับรอง

### 8) พิมพ์รายงานสรุปผล (report-print.php)
- รายงานสรุปตามแผนก/รายงานรายละเอียด พร้อมช่องลงนามผู้เกี่ยวข้อง

### 9) จัดการผู้ใช้งาน (user.php)
- เพิ่ม/แก้ไข/ลบผู้ใช้, เปิด‑ปิดบัญชี
- กำหนด Role (Admin/Webadmin/Inventory) (จากรีวิวเดิม)
- (ฉบับอัปเดต) เพิ่มการกำหนดสิทธิ์ตาม **Site** และรองรับ **AD Integration**

### 10) เปลี่ยนรหัสผ่าน (password.php)
- เปลี่ยนรหัสผ่าน Local + ตรวจสอบความถูกต้อง (ความยาว/ยืนยันรหัสผ่าน)
- (ฉบับอัปเดต) มีแนวทาง/การแจ้งเตือนให้เชื่อมต่อกับ AD (หากใช้งาน AD)

### 11) รายงานวิเคราะห์เชิงลึก (reports.php)
- การ์ดสรุปจำนวน/มูลค่า, วิเคราะห์ตามแผนก
- กราฟเชิงวิเคราะห์ เช่น Pie และ Area แนวโน้มรายเดือน

### หน้าใหม่ (ฉบับอัปเดต): รายงานคะแนน Auditor (auditor-score-report.php)
- สรุปคะแนนเฉลี่ย/Top Auditor + กราฟเปรียบเทียบ
- กรองตาม Site/ปี/ช่วงเวลา
- แยกมิติการให้คะแนน (Accuracy, Completeness, Evidence Quality, Timeliness, Compliance)
- ส่งออก Excel/PDF

## สรุปภาพรวม
ระบบครอบคลุมกระบวนการบริหารทรัพย์สินครบวงจร เหมาะกับองค์กรขนาดกลาง‑ใหญ่ โดยจุดเด่นของเวอร์ชันอัปเดตคือ **Multi‑Site**, **รอบตรวจนับรายปีพร้อม Deadline**, **Import แบบ Incremental**, **AD/SSO**, และ **ระบบให้คะแนน Auditor** ที่ช่วยยกระดับการควบคุมคุณภาพงานตรวจนับและการบริหารงานข้ามโรงงานได้ชัดเจนมากขึ้น
